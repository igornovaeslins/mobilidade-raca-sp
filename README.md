# Mobilidade e raça em São Paulo — prévia

**Onde mora a população negra, o ônibus é mais lento.**

Mapa interativo, por distrito de São Paulo. Clique num bairro para ver: velocidade real
do ônibus, espera extra por irregularidade, % de moradores negros e renda.

➡️ Abra **[index.html](index.html)** (ou a URL do GitHub Pages, quando publicado).

## De onde vêm os dados
- **Olho Vivo / SPTrans** — posição dos ônibus em tempo real, coletada minuto a minuto.
- **Censo 2022 (IBGE)** — composição racial e renda por distrito.

## Honestidade
Amostra **preliminar** (poucos dias de coleta). Leitura **descritiva, não causal**. Parte de
um estudo de mobilidade urbana de São Paulo (Projeto MIIS / Instituto Pólis).

> Prévia temporária para avaliação.
